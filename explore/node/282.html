Module II: Overview of Basic Topics Used in Creating Games

Welcome to Module II: Overview of Basic Topics Used in Creating Games

There are three larger lessons provided in this module. Each lesson covers topics useful to students interested in creating games using App Inventor. Each of the lesson pages provides links to an overview of the topics as well as a corresponding tutorial for students to use to practice the topic or concept they just learned.

  1. Lesson 2.1

    2. Students will learn how to use App Inventor components Canvas, Sprites, and Events and practice their new skills with tutorials linked at the bottom of each topic page

  2. Lesson 2.2

    3. Students will learn how to animate sprites by using Timers and finger motions and how to set the visibility as well as handle collisions between Sprites.

  3. Lesson 2.3 Coming Soon

Be sure to check out the App Inventor Concept Cards for practicing the concepts taught in these lessons in the classroom.
