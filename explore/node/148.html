3. Setup Android Phone (Wireless)

Part 3 of 3 setup instructions

In this section, we'll make sure your phone is ready for use with App Inventor. This involves making sure your phone has the App Inventor Companion App installed, then connecting the phone to your computer through a local wireless (WiFi) network.

We recommend using the wireless connection for live development, but if you do not have access to a reliable wireless (WiFi) connection, or otherwise prefer to connect your computer to your phone with a USB cable please use the Setup Instructions for USB connection (note that Windows users will need to install drivers particular to their phone to connect using a USB cable).

NOTE: If you have trouble connecting using these wireless instructions, make sure your phone is using WiFi and not the cell network. On some devices and locations even with WiFi enabled, the phone continues to use the cell network for Internet connectivity. If this is happening to you, you can disable mobile networking (cell based Internet connectivity) from your device's Android settings menu.


  • Step 1:

    Download and install the App Inventor Companion App* and install it on your phone.

    EITHER download it here: AppInventorCompanionApp.apk and install it on your phone directly (note that to directly install, you will need to allow installation of non-market apps in your device's system settings).

    OR you can access the App Inventor Companion App via the Google Play Store. Scanning or clicking on the QR code below will also take you to the App Inventor Companion App in the Google Play Store.

    *To do live testing of your App Inventor apps with the wireless connection, you must use the latest version of the App Inventor Companion App. If you attempt to use an older version of the Companion App with App Inventor, you will be prompted to update the Companion App.


  • Step 2:

    From the blocks editor, click the "Connect to Device" dropdown, and choose the "WiFi" option. A dialog box with instructions and a 6-letter code will display. A QR code will also be displayed.

  • Step 3:

    Start the App Inventor Companion App on your device (you can launch it like a normal Android application). It will have a text box where you should enter the code displayed in the blocks editor dialog box from step 2*. Once the code is entered, press "Connect to App Inventor"

    *Alternatively, if your phone has a camera, and an appropriate bar code scanning application, you can select the "Scan the QR Code" button on the Debug App and scan the QR code displayed above. If you use the Scan button, you do not have to enter the displayed code or press the "Connect to App Inventor" button.

Setup complete! Run App Inventor for the first time!

  • Run App Inventor
  • Go back to main menu

All you need to get started with MIT App Inventor is a Google account to log in.
