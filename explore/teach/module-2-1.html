Lesson 2.1: Intro to Games and Animations

This module gives basic information about using App Inventor for creating games. Completing the three apps in this module you will help you learn about:

  • Canvas coordinates and properties
  • Timers, Intervals, and Timer-driven events
  • User-driven events (touch, fling, drag, shake)
  • Automatic events (timer, collision)
  • Properties of sprites, making sprites move
